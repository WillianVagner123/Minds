# Documentação conceitual e estratégica

Esta pasta reúne PDFs de referência que explicam a visão, o manual institucional e a proposta de valor da iniciativa **MINDS Performance**.  Embora o código da plataforma resida em outras pastas, estes documentos oferecem o pano de fundo conceitual que inspirou a arquitetura do PINGO.

## Arquivos

| Arquivo | Descrição |
| --- | --- |
| **Minds + Pingo.pdf** | Apresenta a integração entre o módulo de inteligência PINGO e a plataforma MINDS.  Discute como o agente se apoia em dados do humor (BRUMS), carga de treino, contexto e dietas para gerar sinais acionáveis. |
| **Produto Minds Performance (Manual).pdf** | Manual institucional do produto que detalha os objetivos da iniciativa, responsabilidades das partes interessadas e o processo de implementação do PINGO.  Explica como as equipes devem interpretar as notificações e alerta que o sistema gera. |
| **manual_minds_performance.pdf** | Versão completa do manual que combina instruções para treinadores, nutricionistas, psicólogos e equipes técnicas.  Fornece diretrizes sobre a coleta de dados, segurança da informação, fluxos de escalonamento e boas práticas comportamentais. |
| **value_proposition.pdf** | Documento de proposta de valor que apresenta os benefícios do MINDS para atletas e times: integrar dados de performance física e mental, identificar riscos de forma precoce e promover intervenções baseadas em ciência comportamental. |

Estes PDFs complementam a leitura do `README.md` principal do repositório.  Recomenda‑se consultá‑los para contextualizar as regras de pontuação e os fluxos de notificação implementados neste projeto.
